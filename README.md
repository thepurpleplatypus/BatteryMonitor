# BatteryMonitor
Battery Monitor using MQTT.  Based on a Raspberry Pi Zero with an [Automation HAT](https://shop.pimoroni.com/products/automation-hat-mini?variant=31478878077011) from Pimoroni

For 12V DC systems.  Very beta.
